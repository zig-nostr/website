# Built With zig-nostr

Projects built on the `zig-nostr` core. The library is the foundation; the apps
are proof it holds up under real product weight.

## Signet — native remote signer

A NIP-46 bunker for macOS. Your key stays in a local daemon; every signing
request waits for your approval.
→ [`zig-nostr/signet`](https://github.com/zig-nostr/signet) · [read more](/signer)

## The library

- **[`zig-nostr/nostr`](https://github.com/zig-nostr/nostr)** — the protocol
  library itself: keys, events, transport, the local-first store, and the signer
  protocol layer.
